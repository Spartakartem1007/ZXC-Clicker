# ZXC-Clicker


**Кликер про Shadow Fiend, мемы и бесконечный фарм.**

---

## 🎮 Что это

Ты —  ZXC гуль. Ты фармишь души. Каждый клик по СФу приносит демонов в твою голову. Покупай предметы из Dota (Танго, Кларити, Бутылка, Обувь, Аганим, Мидас), чтобы ускорить фарм. Сможешь ли ты достичь предела?


---

## ⚙️ Механики

- **Клик** — приносит демонов. Чем больше кликаешь, тем больше твоя сила.
- **Магазин** — 6 предметов из Dota
- **Авто-клик** — предметы автоматически фармят демонов в фоне.
- **Бесконечная прогрессия** — фарми без ограничений. Ставь рекорды, улучшай предметы, смотри, как растут числа.

---

## 🧪 Версия: Демо

Это демонстрационная версия игры, которая показывает базовую механику кликера с магазином и авто-кликом. Финальный контент, сюжетные события и полноценная концовка будут добавлены в следующих обновлениях.

---

## 🖥️ Как запустить

1. Скачай `.exe` из раздела **Releases**
2. Запусти файл
3. Кликай по картинке СФа и покупай предметы
